TCP

POSIX socket server on port 6379 with one worker thread per client.

Concurrency

Shared map access is protected by mutex-backed datastore operations.

Persistence

SAVE writes a local snapshot that the server loads again at startup.

Expiry

TTL support uses lazy reads plus a background cleanup loop.

Run the C++ server

cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
./build/redisclone
nc localhost 6379
